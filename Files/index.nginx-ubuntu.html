Дипломная работа состоит из трех этапов:

  2. 📁 2. Установка и подготовка Ansible. Установка сервисов
    Настройка nginx, Zabbix, Elasticsearch, Kibana, Filebeat с помощью Ansible.
  3. 📁 3. Проверка ресурсов и выполнение задач
    Финальная проверка, настройка дашбордов Zabbix, Kibana, резервное копирование.
